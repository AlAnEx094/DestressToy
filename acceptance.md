# Acceptance Criteria

## Fidelity
- Section order must match the reference.
- Main layout composition must match the reference.
- Card count must match the reference.
- CTA placement must match the reference.
- Typography hierarchy must feel equivalent to the reference.
- Spacing rhythm must be visually close to the reference.

## Responsiveness
- No layout breakage at 1440px, 1024px, 768px, 390px.
- Hero must remain readable on mobile.
- Card grids must collapse cleanly.

## Implementation
- Clean component structure.
- No obvious placeholder visual hacks.
- No broken states.
- No overflowing content.
- No mismatched paddings caused by arbitrary Tailwind additions.

## Non-goals
- Perfect pixel-level parity is not required.
- Creative reinterpretation is not allowed.
